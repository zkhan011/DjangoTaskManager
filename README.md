# DjangoTaskManager

This is simple task assignement and management portal created using Django framework and postgres database.

Installation is fairly simple
Step 1: Download the zip
Step 2: on webserver dump all the module files in htdocs folder and static files in static folder
Step 3: Create a database and provide its credential in manage.py
Step 4: open Linux command line and type python manage.py from the root folder
Step 5: once all is done restart the webserver and the app will be running.
